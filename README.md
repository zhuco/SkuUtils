# SKU 计算器 Web 版

当前项目已按 `SKU计算器Web版开发文档.md` 重构为纯前端 SPA：

- 无需登录
- 商品和历史默认只保存在当前浏览器
- 计算逻辑在前端本地执行
- 适合静态部署到公网

## 当前技术栈

- React + TypeScript + Vite
- Ant Design
- Dexie + IndexedDB
- Zustand
- Zod
- Decimal.js
- Vitest

## 已实现能力

- 商品基础信息录入
- 默认规格与自定义规格管理
- 重量区间运费规则
- 固定值 / 百分比费用模式
- 批量规格测算
- 最近商品
- 历史记录与参数快照
- 本地 JSON 导入导出
- CSV 结果导出

## 启动方式

开发模式：

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

本地预览：

```bash
npm run preview
```

默认预览地址：

- `http://127.0.0.1:4173`

## 手动测试建议

建议至少验证以下流程：

1. 首次打开页面后默认规格、默认运费、默认设置自动初始化。
2. 输入商品名称、单件成本、单件重量后执行“仅计算”。
3. 执行“添加并保存”后，最近商品和历史记录正确生成。
4. 在设置中修改费用参数、增加自定义规格并重新计算。
5. 导出本地 JSON，清空商品或历史后重新导入。

## 部署说明

这是静态前端项目，不依赖服务端数据库。适合部署到：

- Vercel
- Cloudflare Pages
- Netlify
- 任意 Nginx / Caddy 静态站点

## 数据说明

业务数据默认写入当前浏览器的 IndexedDB。请注意：

- 更换设备不会同步
- 清理浏览器站点数据会导致本地数据丢失
- 建议定期导出 JSON 备份
